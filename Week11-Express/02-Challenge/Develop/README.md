# Note Taker

## Description
The application enables users to create and save notes to log tasks and reminders.

## Installation
Open the deployed index.html file and follow the prompts to create and save your own notes.

## Built With
- JavaScript
- JSON
- Node.js
- Express.js
- Heroku
- Visual Studio Code
